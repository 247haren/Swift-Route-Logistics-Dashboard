# 📌 Project Overview

SwiftRoute Logistics Dashboard is a comprehensive Business Intelligence solution developed using **Microsoft Power BI** to help logistics companies monitor operational performance across deliveries, drivers, hubs, and vehicles.

The dashboard transforms raw logistics data into meaningful insights through interactive visualizations, KPI tracking, and dynamic filtering, enabling stakeholders to make informed, data-driven decisions.

---

# 🎯 Business Problem

Managing logistics operations involves coordinating deliveries, drivers, hubs, and vehicles efficiently. Without centralized reporting, businesses struggle to identify operational bottlenecks and optimize performance.

This dashboard addresses these challenges by providing:

- 📦 Delivery Performance Monitoring
- 🚚 Fleet Utilization Analysis
- 👨‍✈️ Driver Performance Evaluation
- 🏢 Hub Efficiency Tracking
- ⭐ Customer Satisfaction Monitoring
- 📈 Executive KPI Reporting

# 📈 Key Performance Indicators

| KPI | Description |
|------|-------------|

| ⏰ On-Time Delivery Rate | Percentage of orders delivered on time |
